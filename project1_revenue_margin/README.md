# Revenue & Gross Margin Analysis

## Objective
Analyze revenue and profitability by SKU, category, and sales channel.

## Tables Used
orders, order_items, products

## Key Metrics
- Revenue
- Cost
- Gross Margin
- Gross Margin %

## SQL Skills Demonstrated
- JOIN
- GROUP BY
- CTE
- Window Functions

## Key Findings
- Toys generated the highest revenue.
- Electronics had the highest gross margin percentage.
- Amazon delivered strong sales but lower margins.
